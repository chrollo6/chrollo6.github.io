💡 Section 5: Bonus: Essential Mac Tips for Windows Users
  • CMD = CTRL (in most shortcuts)
  • No "maximize" — green button toggles full screen (not resize)
  • Menu bar is global (top of screen), not per window
  • CMD + Q quits apps; red X only hides them
  • DMG files are like virtual CDs — you can eject them

🛠️ Section 6: Useful macOS Terminal Commands
  • caffeinate -diu — Prevent sleep (display, idle, user activity) while running
  • open . — Open the current folder in Finder
  • pbcopy < file.txt — Copy a file’s contents to clipboard
  • pbpaste > file.txt — Paste clipboard contents into a file
  • say "hello" — Make your Mac talk (fun for testing audio output)
  • top — See system processes (like Task Manager)
  • uptime — Shows how long the system has been running
  • tmutil listlocalsnapshots / — List Time Machine snapshots (if enabled)
  • sudo shutdown -h now — Shutdown your Mac immediately

📋 Section 7: Clipboard & Copy Tips
  • Universal Clipboard: Copy on Mac, paste on iPhone/iPad (requires iCloud + Bluetooth)
  • pbcopy and pbpaste — Use Terminal to access clipboard
  • echo "text" | pbcopy — Copy string to clipboard
  • pbpaste | grep "keyword" — Pipe clipboard into other tools
  • Clipboard is system-wide: works between Terminal, Notes, and GUI apps

⌨️ Section 9: Hidden Keyboard Gems
  • CMD + ` — Cycle windows within the same app
  • CMD + H — Hide current app instantly
  • CMD + W — Close current tab or window
  • CTRL + CMD + Q — Lock your Mac immediately
  • CMD + Shift + . — Show/hide hidden files in Finder
  • CMD + Option + Esc — Force quit window (like Task Manager)

🧾 Section 10: Aliases and .zshrc

Aliases let you create shortcuts for terminal commands. They save keystrokes and reduce errors for repetitive tasks.

.zshrc is the config file for the Z shell (default on macOS). You can define aliases, environment variables, and other settings there.

Example Aliases:

alias ll="ls -la"
alias reload="source ~/.zshrc"
alias whisper="whisper --model medium --language en"

How to add:

  1. Run nano ~/.zshrc
  2. Add your aliases at the bottom
  3. Save with CTRL + O, then exit with CTRL + X
  4. Run source ~/.zshrc to apply changes

You can also export environment variables or tweak prompt settings here.
